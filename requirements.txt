# 心理咨询模拟系统 - 部署环境要求

## 系统要求

### 硬件要求
- **CPU**: 双核 2.0GHz 或更高
- **内存**: 4GB RAM 或更高
- **存储空间**: 至少 1GB 可用空间
- **网络**: 稳定的互联网连接

### 软件环境

#### 操作系统支持
- Windows 10/11 (64位)
- macOS 10.14 或更高版本
- Ubuntu 18.04 或更高版本
- 其他主流 Linux 发行版

#### 运行时环境
- **Node.js**: 14.0.0 或更高版本
- **npm**: 6.0.0 或更高版本 (通常随 Node.js 安装)

#### 浏览器要求
- Google Chrome 80+
- Mozilla Firefox 75+
- Safari 13+ (仅 macOS)
- Microsoft Edge 80+

#### 网络要求
- 稳定的互联网连接
- 支持HTTPS协议
- 防火墙允许 3000 端口通信

## 依赖包清单

### Node.js 生产依赖
```
express >= 4.18.0
cors >= 2.8.0
http-proxy-middleware >= 2.0.0
node-fetch >= 2.7.0
```

### 开发和测试依赖 (可选)
```
nodemon >= 2.0.0
```

## API 外部依赖

### Dify AI 平台要求
- **来访者工作流**: 具备聊天功能的 Dify 应用
- **督导工作流**: 具备聊天功能的 Dify 应用
- **API 访问权限**: 已启用 API 访问功能
- **网络连接**: 能够访问 dify.ai-role.cn 域名

## 安全要求

### 网络安全
- 系统需要在安全的网络环境中运行
- API 密钥需要妥善保管，不要在版本控制系统中提交
- 建议使用企业防火墙保护

### 数据隐私
- 所有对话数据仅存储在本地浏览器中
- 系统不会上传任何用户数据到第三方服务器
- 需要定期清理浏览器缓存以保护隐私

## 安装验证

### 环境检查命令
```bash
# 检查 Node.js 版本
node --version

# 检查 npm 版本
npm --version

# 检查网络连接
ping dify.ai-role.cn
```

### 功能测试
- 安装完成后运行 `npm start`
- 访问 `http://localhost:3000/index.html`
- 确认系统能够正常启动和使用

## 性能要求

### 响应时间
- 页面加载时间 < 3秒
- API 响应时间 < 10秒
- 界面交互响应 < 1秒

### 并发支持
- 支持至少 5 个用户同时使用
- 支持长时间连续运行 (8小时+)

## 维护要求

### 定期维护
- 每月检查 Node.js 版本更新
- 每季度更新依赖包
- 定期检查 API 密钥有效性
- 定期清理日志文件

### 备份要求
- 定期备份配置文件
- 导出重要的用户学习记录
- 保存 API 密钥的安全备份
